Automação de Cadastro de Produtos com Python (RPA)

Sobre o projeto

Este projeto consiste em uma automação desenvolvida em Python para realizar o login em um sistema web e cadastrar automaticamente produtos a partir de uma base de dados em CSV.

A automação simula ações humanas como cliques, digitação e navegação na interface do sistema, aplicando conceitos de **RPA (Robotic Process Automation)** para eliminar tarefas manuais repetitivas.

---

Tecnologias utilizadas
- Python 3
- PyAutoGUI
- Pandas
- OpenPyXL

---

Funcionalidades
- Abertura automática do navegador
- Login automatizado em sistema web
- Leitura de base de dados em arquivo CSV
- Cadastro automático de produtos
- Tratamento de campos opcionais
- Execução repetitiva até o fim da base de dados
